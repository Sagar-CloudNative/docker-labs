Hands-on Docker labs to go along with the **TechOpsTutorials Docker playlist** on YouTube.

Each lab folder matches a video in the playlist and includes commands, YAML files, and everything you need to follow along.

---

## 🎯 Who This Is For

- Developers and DevOps engineers getting started with Docker
- Anyone moving from VMs to containers
- Engineers preparing for real-world containerization projects

---

## ⚙️ Prerequisites

- A Linux system (Rocky Linux, Ubuntu, or similar)
- Basic command line knowledge
- No prior Docker experience needed — we start from scratch

## 🗂️ Repo Structure

```
docker-labs/
├── README.md
├── LICENSE
├── 01-containers-and-orchestration/
│   └── README.md
├── 02-docker-installation/
│   ├── README.md
│   └── commands.md
├── 03-docker-images-containers/
│   ├── README.md
│   └── commands.md
└── ...
```

Each lab folder contains:
- `README.md` — concept overview, what you'll learn, demo summary
- `commands.md` — all commands used in the video
- YAML or config files where needed

---

## 🚀 How to Use This Repo

```bash
git clone https://github.com/Sagar-CloudNative/docker-labs.git
cd docker-labs
```

Then open the folder for the video you are watching and follow along.
